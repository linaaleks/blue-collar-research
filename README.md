# Рынок высококвалифицированных синих воротничков в РФ

Исследование для оценки потенциала рынка, барьеров и финансовой модели запуска образовательного направления.

## Содержимое репозитория

| Файл | Описание |
|------|-----------|
| **index.html** | Интерактивный отчёт для стейкхолдеров (открой в браузере) |
| **SYNTHESIS.md** | Полный синтез по 4 исследовательским блокам |
| **SUMMARY.md** | Краткое резюме (Executive Summary) |
| **TASK.md** | Исходное задание на исследование |
| **Claude_Research.md**, **OpenAI_DeepResearch.md**, **Gemini_DeepResearch.md**, **Perplexity_Research.md** | Отчёты первой волны (4 агента) |
| **Claude_Research_2.md**, **OpenAI_DeepResearch_2.md**, **Gemini_DeepResearch_2.md**, **Perplexity_Research_2.md** | Отчёты от коллег (4 агента) |
| **_work/** | Нормализованные отчёты и справочники кейсов/источников |

## Источники данных

- **8 отчётов:** 4 первой волны + 4 от коллег (аналогичные AI-агенты, независимый запуск).
- Синтез объединяет все данные по структуре: контекст и цель → рынок труда → технологии → регулирование → финансовая модель.

## Как открыть отчёт

1. Локально: открыть `index.html` в браузере.
2. После публикации на GitHub Pages: ссылка вида `https://<username>.github.io/blue_collar_research/`.
